# SMSOnline - Virtual Number Marketplace

A Next.js application for buying and managing virtual phone numbers.

## Features
- **Landing Page**: Overview of services.
- **User Dashboard**: Manage purchased numbers, view history.
- **Admin Dashboard**: Overview of platform statistics and users.

## Project Structure
- `app/page.tsx`: Main landing page.
- `app/dashboard/page.tsx`: Authenticated user area.
- `app/admin/page.tsx`: Administration panel.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
